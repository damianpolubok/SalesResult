# Aplikacja do analizy wyników sprzedaży (import z csv, export do Excel)
Projekt na Zaawansowane programowanie w języku Python

Damian Połubok, Jakub Hryniewicki, Łukasz Dudicz

Python 3.12

Inside project folder e.g. `C:\Users\user\repos\SalesResult\SalesResult>` run:
```powershell
py -3.12 -m venv .venv; .\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

If you want to run it through PowerShell, inside project folder e.g. `C:\Users\user\repos\SalesResult\SalesResult>` run:
```powershell
.\.venv\Scripts\python.exe SalesResult.py
```
